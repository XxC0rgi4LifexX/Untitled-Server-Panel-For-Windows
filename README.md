# 🚀 Untitled Server Panel (Windows Edition)

A lightweight server panel built with **Node.js**, designed to simplify server management while also supporting **browser-based access mode**.

This project replaces the usual cluttered console experience with something more visual, structured, and easier to control.

---

## ✨ Features

- ⚙️ Built with Node.js for performance and flexibility  
- 🌐 Browser Mode support for web-based control  
- 🖥️ Windows-friendly server execution  
- 📦 Simple setup with minimal dependencies  
- 🔌 Extendable architecture for custom tools & modules  
- 📊 Clean server output and management flow  

---

## ⚡ Installation

### 1. Clone the repository
```bash
git clone https://github.com/XxC0rgi4LifexX/Untitled-Server-Panel-For-Windows.git
cd Untitled-Server-Panel-For-Windows
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the server
```bash
node index.js
```

or:

```bash
npm start
```

---

## 🌐 Browser Mode

Once running, open:

```
http://localhost:3000
```

(Port may vary depending on config)
